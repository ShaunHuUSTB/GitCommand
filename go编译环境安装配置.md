# 安装go

1. ​**访问官网**​：

* ​**官方地址**​：[https://go.dev/dl/](https://go.dev/dl/?spm=5176.28103460.0.0.24b86308LqvwUz)
* ​**国内镜像（推荐，速度更快**​：[https://golang.google.cn/dl/](https://golang.google.cn/dl/?spm=5176.28103460.0.0.24b86308LqvwUz)

2. 下载安装go安装包，并安装
3. 配置环境变量并添加到PATH：GOROOT, GOPATH,GOPROXY，也可在IDE的设置中配置

# 安装make(可选)

1. 安装mingw64或cygwin
2. 添加安装路径\bin到PATH
